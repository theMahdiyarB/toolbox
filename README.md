# 🧰 جعبه‌ابزار آفلاین

**بیش از ۹۰ ابزار کاربردی — کاملاً آفلاین، بدون نیاز به اینترنت، قابل نصب روی موبایل و دسکتاپ.**

🌐 **پیاده‌سازی‌شده:** [mahdiyar.info](https://mahdiyar.info/)

---

## ✨ ویژگی‌ها

- **کاملاً آفلاین** — Service Worker تمام فایل‌ها را در اولین بازدید کش می‌کند؛ پس از آن بدون اینترنت کار می‌کند
- **قابل نصب (PWA)** — به صفحه اصلی اندروید/iOS اضافه می‌شود یا به‌عنوان اپ دسکتاپ نصب می‌شود
- **معماری تک‌فایلی** — تمام اپلیکیشن در یک `index.html` است (~۶۶۰KB)؛ بدون نیاز به build، npm یا فریمورک
- **Material Design 3** — تم تاریک/روشن، چیدمان RTL، فونت وزیرمتن، بازخورد لمسی در موبایل
- **فارسی‌محور** — رابط کاربری کامل فارسی، تقویم جلالی، محتوای مخصوص ایران
- **حریم خصوصی کامل** — تمام پردازش‌ها در مرورگر انجام می‌شود؛ هیچ داده‌ای به سرور ارسال نمی‌شود

## 🚀 راه‌اندازی

### روش ۱ — باز کردن مستقیم فایل

فایل `index.html` را دانلود کرده و در هر مرورگر مدرنی باز کنید. اکثر ابزارها از طریق `file://` کار می‌کنند.

> ⚠️ برخی قابلیت‌ها مانند Service Worker و PDF.js به یک سرور HTTP واقعی نیاز دارند. روش ۲ را ببینید.

### روش ۲ — سرور محلی (توصیه‌شده)

هر سرور استاتیکی کافی است. ساده‌ترین گزینه‌ها:

```bash
# پایتون (داخلی)
python3 -m http.server 8000

# Node.js
npx serve .
```

سپس آدرس `http://localhost:8000` را در مرورگر باز کنید.

### روش ۳ — استقرار روی GitHub Pages

راهنمای کامل را در بخش بعدی ببینید.

## 🌐 استقرار روی GitHub Pages

### مرحله ۱ — ساخت ریپوزیتوری

۱. به [github.com/new](https://github.com/new) بروید
۲. نام ریپو را وارد کنید (مثلاً `toolbox`)
۳. دسترسی را روی **Public** بگذارید
۴. **هیچ‌کدام** از گزینه‌های Add README، .gitignore یا License را تیک نزنید
۵. روی **Create repository** کلیک کنید

### مرحله ۲ — ارسال فایل‌ها

```bash
# در پوشه پروژه
git init
git add .
git commit -m "اولین کامیت — جعبه‌ابزار آفلاین"
git branch -M main
git remote add origin https://github.com/USERNAME_SHOMA/toolbox.git
git push -u origin main
```

### مرحله ۳ — فعال کردن GitHub Pages

۱. در ریپو به **Settings** → **Pages** بروید
۲. زیر **Source** گزینه **Deploy from a branch** را انتخاب کنید
۳. شاخه: `main`، پوشه: `/ (root)`
۴. روی **Save** کلیک کنید
۵. پس از حدود ۶۰ ثانیه سایت شما در این آدرس در دسترس است:
   `https://USERNAME_SHOMA.github.io/toolbox/`

### مرحله ۴ — دامنه اختصاصی (اختیاری)

اگر دامنه‌ای دارید (مثلاً `tools.example.com`):

۱. در پنل DNS خود یک رکورد `CNAME` اضافه کنید:
   `tools.example.com → USERNAME_SHOMA.github.io`
۲. در تنظیمات GitHub Pages دامنه سفارشی را وارد کنید
۳. گزینه **Enforce HTTPS** را فعال کنید
۴. در `manifest.json` مقادیر `start_url` و `scope` را به آدرس جدید تغییر دهید

## 🤝 مشارکت در پروژه

از مشارکت شما استقبال می‌کنیم. برای افزودن ابزار جدید:

۱. ریپو را Fork کنید
۲. HTML کارت ابزار را داخل `<!-- /tool-container -->` در `index.html` اضافه کنید
۳. جاوااسکریپت ابزار را در بلوک `<script>` اصلی اضافه کنید
۴. یک لینک‌کارت در بخش مناسب صفحه اصلی اضافه کنید
۵. یک آیتم در منوی کشویی `<nav>` اضافه کنید
۶. نام ابزار را به `TOOL_TITLES` اضافه کنید
۷. یک Pull Request باز کنید

لطفاً سبک کد را رعایت کنید — JavaScript خالص، بدون فریمورک، سازگار با RTL، قابل استفاده آفلاین.
